`NoSQL vs SQL` 
1. `SQL` called Relational Databases. Table based using number of rows and have a standard schema (structured).
SQL is scalable vertically and can be increase load by increaseing CPU, RAM, SSD.

1. `NoSQL` non-relational database. Document based, a collection of key value pair  which have no standard schema(Unstructured)
NOSQL is scalable horizontally and you can increase load by increaseing the # of servers.
